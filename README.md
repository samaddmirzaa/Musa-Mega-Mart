# 🛒 Musa MegaMart — Retail Sales Analysis


---

## 📋 Project Overview

This project is a data analyst work on **Musa MegaMart**, a multinational retail corporation, and we have been handed a sample dataset from a retailer they're evaluating for acquisition. The goal is to dig into the sales history, understand the business's strengths, and assess whether their discount strategy can attract customers without eroding margin.

---

## 🎯 Objectives

- **Ingest** multiple flat files efficiently with optimized dtypes
- **Join** tables into a single unified source of truth
- **Shape & aggregate** sales data to calculate meaningful KPIs
- **Visualize** findings to communicate insights clearly

---

## 📁 Project Structure

```
├── 01_DataFrames.ipynb       # DataFrame fundamentals & exploration
├── 02_data_viz.ipynb         # Visualization with matplotlib/seaborn
├── 03_project1.ipynb         # Project 1: Initial sales analysis
├── 04_time_series.ipynb      # Time series analysis & trends
├── 05_import_export.ipynb    # Optimized data ingestion & export
├── 06_combining_dfs.ipynb    # Joining & merging DataFrames
├── 07_project2.ipynb         # Project 2: KPIs & discount analysis
└── README.md
```
